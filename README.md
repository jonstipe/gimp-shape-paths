# gimp-shape-paths
A set of Script-Fu to create geometric shapes in GIMP.

Now supports vector layers in GIMP 3.2!

Installation
============

Download the file "shape-path.scm" into your GIMP scripts folder.

Supported shapes
================
* Rectangle
* Square
* Circle
* Ellipse
* Oval
* Rounded rectangle
* Advanced rounded rectangle
* Polygon
* Star
* Flowers & spikes
* Gear
* Triangle wave
* Square wave
* Sine wave
* Archimedean spiral
